# Attendance Management System

## Project Description
The Attendance Management System is a simple Python-based console application that allows a user to record and display student attendance. The system enables the user to enter student names, mark attendance as Present or Absent, and generate an attendance report.

This project was developed following the Software Development Life Cycle (SDLC).

---

## Features
- Date-based attendance tracking
- Student ID and name management
- Search attendance by student ID
- Attendance percentage calculation
- Save attendance records to a file
- Menu-driven interface

---

## Technologies Used
- Python (no external libraries)
- Visual Studio Code
- Git & GitHub
